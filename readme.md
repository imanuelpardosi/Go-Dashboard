# Go Dahsboard

## Instalation

```
Download or Clone Go-Dashboard application:
$ git clone https://github.com/imanuelpardosi/Go-Dashboard.git

Open Git Bash on the project, then:
$ composer install
$ php artisan vendor:publish
$ php artisan migrate
$ php artisan key:generate

Running on :
OS : Windows 10
Framework : Laravel 5.1
Database : MySQL
Server : Apace (Xampp)
PHP Unit : 4.8.27
